# Pennsylvania First-Time Homebuyer Landing Page

Create this file:

`.env.local`

Paste this exact line into it:

`RESEND_API_KEY=your_real_resend_api_key`

Run locally:

`npm run dev`

Open:

`http://localhost:3000`

Deploy by uploading to Vercel (or any Node host) and setting the same `RESEND_API_KEY` environment variable there.
